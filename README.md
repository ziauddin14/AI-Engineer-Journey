## 📌 Roadmap Philosophy

Poori tree mein 100+ topics hain, lekin har topic ko equal depth dena zaroori nahi. Isliye maine roadmap ko **3 levels** mein divide kiya hai:

- 🟢 **Level 1 — Must Master** (Core foundation, deeply seekhna hai)
- 🟡 **Level 2 — Working Knowledge** (Kaam chalane layak samajh honi chahiye)
- 🔵 **Level 3 — Specialization** (Zarurat aur interest ke mutabiq)
